7º Loopings
Looping ou laço de repetição, é  ultilizado para interar sobre algo que está fazendo, para tomar uma ação.

while - Enquanto algo for verdade ele vai continuar executando.
for - A partir de certo ponto ele vai incrementar tomando alguma ação 
forin - Ele vai integrar em cima de um array ou uma função e devolver um índice
forof - Ele também vai integrar mas devolvendo um objeto.

Ex : for
 for (var index = 0; index <= multiplicador; index++){
    document.write(número + "X" + index + "=" + (número * index) + "<br>");
    
}

Função - é um método para realizar o que pode isolar responsabilidades e funções.

Ex: 
function CalcularTaboada(número,multiplicador) {
    for (var index = 0; index <= multiplicador; index++){
        document.write(número + "X" + index + "=" + (número * index) + "<br>");
}
    
}
